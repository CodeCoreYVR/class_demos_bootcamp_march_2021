Flex Properties

With flex, you'll discover that far less CSS is required to achieve certain layouts

The flex container behaves like a block element for the most part. It'll take up the entire width available by default. It's its children, flex items, that have special behaviour. Think of the flex container as the choreographer and flex items, its dancers.
